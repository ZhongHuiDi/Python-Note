## 02.1 Linux基础命令
### 02.1.1 显示日期的指令：date
* date：显示当前日期
```
[root@i-jjr0xl1a /]# date
Fri Dec 29 14:17:02 CST 2017
```
* date +%Y%M%D：只显示当前年月日
```
[root@i-jjr0xl1a /]# date +%Y%M%D
20171912/29/17
```
* date +%H:%M:%S：只显示当前时分秒
```
[root@i-jjr0xl1a /]# date +%H:%M:%S
14:21:28
```
