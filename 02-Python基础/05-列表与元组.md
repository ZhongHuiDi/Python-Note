## 4.1 数据结构
* 数据结构就是可以处理的一些数据
* python中有4种数据结构，分别是：列表、元组、字典和集合

## 4.2 列表
### 4.2.1 列表、索引的定义
* 列表是一组有序项目的数据结构，项目内可以放任何数据类型，每个项目用逗号分隔。
* 索引：列表是一组有序的项目，所有的元素(从左到右)都是有编号的----从0开始递增，我们将这些编号成为索引，元素可以通过索引值访问

```python
list = [1,2,3,4]
list[0]
1
```


### 4.2.1 列表的基础操作
* 创建
* 查询
* 切片
* 增加
* 修改
* 删除
* 循环
* 排序

#### 1. 创建
* 方法一：直接创建
  * L1 = [] #定义空列表
  * L2 = ['a','b','c']
  * L3 = ['abc',['def','ghi']] #嵌套列表

```python
L1 = [] #定义空列表
L2 = ['a','b','c'] #存4个值，索引为0-3
L3 = ['abc',['def','ghi']] #嵌套列表
```

* 方法二：通过字符串创建

```
L4 = list('Hello')
['H','e','l','l','o']
```

#### 2 查询
* 通过索引值查询: L[index] #index代表索引值
* 注意：
  * 如果index>0，则为从左到右查询
  * 如果index<0，则为从右到左查询
  * .index()可以查出特定值的索引值[如果有重复的值，那么会从左到右查找第一个的索引值]

```python
L2 = ['a','b','c','d','a','e',1,2]
L2[2] #通过索引取值
'c'
L2[-1] #通过索引从列表后边开始取值
2
L2[-2] #通过索引从列表后边开始取值
1
L2.index('a') #返回指定元素的索引值，从左右查找，找到第一个匹配的值的索引
0
L2.count('a') #统计指定元素的个数
2
```


#### 3 切片
* 切片就是只取列表的一部分,默认规则从左到右
* 切片的只包含开头不包含结尾，典型的“顾头不顾尾”
* 常用的切片取法
  * 取前n个值，L[0:n] 或者 L[:n]
  * 取最后n个值，L[-n:]
  * 取全部：L[:] 或者 L
  * 跨序取数：L[m:n:a] #其中a为步长

```python
L2 = ['a','b','c','d','a','e',1,2]
#取前3个：
L2[0:3]
L2[:3]
#取最后五个：
L2[-5:-1] (错误，没有包含最后一个)
L2[-5:0] (错误，默认从左往右)
L2[-5：] (正确)
#取全部：
L2[:]
L2
#跨顺序取数
L2[:7:2] #2为步长，代表取数跨n-1个
```

#### 4 修改与插入
* L.append(value)  #追加，在最后增加

```python
L2.append('peiqi') #将peiqi加在最后面
```

* L.insert(index,value) #插入value，index为选择插入的位置

```python
L2.insert(0,'peiqi') #将peiqi插入在第0个索引值
```

* L[index] = new_value #将索引值为index的值修改为new_value

```python
L2[3] = 'maomi'
```

* 注意：批量修改，可以加上切片器,但是一般都不这么修改，因为会把修改的内容一个一个插入进去。


#### 5 删除
* n2.pop() #删除最后一个,会返回最后一个元素
* n2.remove('') #删除指定的元素，如果包含多个重复的元素，默认只删除从左到右第一个
* del L2[2] #删除指定索引值的元素，del是全局性的删除，想删除谁就删除谁，del可以批量删除

#### 6 循环
* for i in L2： #循环取数
* for循环语法：
  * 列表内循环：for i in L2:
  * 如果没有列表的循环：for i in range(n) #range自动生成0-(n-1)之间的列表

```
L2 = ['a','b','c','d','e','f','g']
for i in L2:
    print(i)
```

* while循环和for循环的区别：
    * while循环可以是死循环，for循环是有边界的

#### 7 排序
* 列表永远都是有序的，因为列表按照索引取值，所以一直都是有序的
* L.sort() #正向排序
  * 注意：字符和数字不能一起排序
  * 排序规则：按照ASCII码表的顺序
* L.reverse() #反向排序

#### 8 相加
* L + M #直接相加,L、M都是列表
* L.extend(M) #用extend相加

#### 9 清除
* L.clear() #清空整个列表

#### 10 拷贝
* L.copy() #拷贝L列表
* 注意拷贝列表的时候，是在不同的文件路径，也就是说原列表改变，不影响拷贝的列表。

#### 11 注意
* 列表和变量的相等的赋值不一样，变量指定的是具体的值，一个变量变，不影响另外一个相等的变量的值，而列表如果变，其余也跟着变
* 使用copy可以让两个列表独立，和变量一样

### 附加1：小技巧(enumerat()函数)
* 之前已经学习了用print打印出来列表里所有的值，现在我们学习一个更简单的方法：enumerates函数
  * 记序列数方法：enumerates(L2)
* 如果对一个列表，既要遍历索引又要遍历元素时，首先可以这样写：

```python
list1 = ["你", "好", "Python", "Class"]
for i in list1:
    print(list1.index(i),i)

#输出如下：
0 你
1 好
2 Python
3 Class
```

* 如果用enumerate函数来写则会方便很多，如下：

```python
list1 = ["你", "好", "Python", "Class"]
for index, item in enumerate(list1): #index代表索引值，item代表里面的元素
    print (index, item)

#输出如下：
0 你
1 好
2 Python
3 Class
```

* enumerate还可以接收第二个参数enumerate(L,n)，n用于指定索引起始值，如：

```python
list1 = ["你", "好", "Python", "Class"]
for index, item in enumerate(list1,1):
    print (index, item)

#输出如下：
1 你
2 好
3 Python
4 Class
```

* enumerate还可以统计文件的行数

```python
count = 0
for index, line in enumerate(open(filepath,'r'))：
    count += 1
```


### 附加2：计算列表长度(len)
* len(L)  #L代表列表

### 附加3：查看列表的内存地址(id)
* id(L) #查看L的内存地址
  * 变量的内存地址：
    * a = b = 1 ,id(a) = id(b)  #a、b代表变量
    * 如果a = 2，则 id(a) <> id(b)
  * 列表的内存地址:
    * L = M ，id(L) = id(M) #L、M代表列表
    * 如果L变化，则id(L) = id(M)

### 附加4：copy用法
* 浅copy:只是复制了值，而内存地址不同
* 深copy:不仅复制了值，而且内存地址还相同
  * copy.deepcopy()

## 4.3 元组
### 4.3.1 元组的定义
* 列表和元组的区别在于，列表可以修改，而元组不能修改。
* 元组由圆括号()和逗号分隔项目组成

### 4.3.2 创建元组
* 创建元组

```python
(1,2,3)
```

* 创建空元组

```python
()
```

* 创建一个值的元组:必须加个逗号，即使只有1个值

```python
print(3*(40+2))
#输出
126

print(3*(40+2,))
#输出
(42, 42, 42)
```

### 4.3.3 tuple函数
* tuple函数与listh函数基本上是一样：以一个序列作为参数并把它转换为元组

```python
tuple([1,2,3])
#输出
(1, 2, 3)
tuple('Hello')
#输出
('H', 'e', 'l', 'l', 'o')
```

### 4.3.4 基本元组操作
* 元组可以创建、访问元素、切片，元组切片后还是元组，方法类似。
